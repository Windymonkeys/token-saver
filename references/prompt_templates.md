# Prompt 模板库

预定义常用 Prompt 模板，减少重复描述，提高效率。

---

## 使用方式

复制模板，替换 `{参数}` 即可使用。

---

## 1. 代码相关

### 代码审查
```
请审查以下 {language} 代码：

检查项目：
- 安全性漏洞
- 性能瓶颈
- 代码规范
- 可读性

代码：
{code}

请按严重程度（高/中/低）分类列出问题。
```

### 代码生成
```
请用 {language} 编写一个 {功能描述}：

要求：
- 输入：{输入格式}
- 输出：{输出格式}
- 约束：{约束条件}

请包含注释和示例用法。
```

### 代码解释
```
请解释以下代码的功能：

{code}

要求：
- 逐行解释
- 说明核心逻辑
- 列出依赖项
```

### Bug 修复
```
以下代码有问题：

代码：
{code}

错误信息：
{error}

期望行为：
{expected}

请：
1. 分析原因
2. 提供修复方案
3. 解释修改原因
```

## 模板参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `{language}` | 编程语言 | Python, JavaScript |
| `{code}` | 代码片段 | 函数/类/文件内容 |
| `{data}` | 数据内容 | JSON/CSV/表格 |
| `{text}` | 文本内容 | 任意文本 |
| `{requirements}` | 需求描述 | 功能需求列表 |
| `{format}` | 格式要求 | JSON/Markdown/表格 |

---

## 快速使用

```python
# Python 示例
template = """
请审查以下 {language} 代码：
{code}
"""

prompt = template.format(
    language="Python",
    code="def hello(): print('world')"
)
```
